# Bring the Firm

Standalone SvelteKit app for the Bring the Firm builder.

The frontend connects to the shared Overbase Convex deployment with `PUBLIC_CONVEX_URL`.
The runtime endpoints are called by Overbase server-side code and require `OVERBASE_BUILDER_APP_SECRET`.

## Commands

```sh
npm install
npm run dev
npm run check
npm run build
```
